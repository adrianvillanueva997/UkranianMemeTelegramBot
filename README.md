# UkranianMemeTelegramBot
> Telegram [bot](https://t.me/xiao_bot) with tons of useful commands

![](ukranianmemepic.jpg)

## Installation

Just make sure you have python 3.7x or more and install the following:
```
pip3 install -r requirements.txt
```
